5. 设置系统的启动级别

Linux分为7个启动级别：

  • 0 - halt (Do NOT set initdefault to this)
  • 1 - Single user mode
  • 2 - Multiuser, without NFS (The same as 3, if you do not havenetworking)
  • 3 - Full multiuser mode
  • 4 - unused
  • 5 - X11
  • 6 - reboot (Do NOT set initdefault to this)

翻译一下：

  • 0 系统停机状态（不能设置为默认级别）
  • 1 单用户工作状态
  • 2 多用户状态（没有NFS）
  • 3 多用户状态（有NFS）（命令行）
  • 4 系统未使用，留给用户
  • 5 图形界面
  • 6 系统正常关闭并重新启动（不能设置为默认级别）
