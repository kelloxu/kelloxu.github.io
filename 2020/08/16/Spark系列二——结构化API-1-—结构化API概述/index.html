Spark系列二——结构化API(1)—结构化API概述
发表于 2020-08-16| 更新于 2020-08-16|spark
字数总计:191|阅读时长: 1 分钟
| 阅读量:
Spark 是一个分布式编程模型，用户可以在其中指定转换操作，通过多个转换构成一个指令的有向无还图。指令的图的执行过程作为一个作业由一个动作操作触发，在执行过程中一个作业被分解成多个阶段和任务在集权上执行。转换操作和动作操作操纵的逻辑结构是 DataFrame 和 DataSet，创建一个新的 DataFrame 或 DataSet 需要调用一个转换操作。如果要启动计算，或者将 DataFrame 或 DataSet 转换为本地语言的类型，则需要调用一个动作。
